## 内容创作

### 资讯类型
1. **每日简报**：当天重要 AI 新闻汇总
2. **深度分析**：重要事件的详细解读
3. **产品评测**：新 AI 工具的试用体验
4. **趋势观察**：AI 技术发展趋势分析

### 内容风格
- **及时性**：快速报道最新动态
- **准确性**：确保信息来源可靠
- **深度性**：不只是转述，要有分析
- **实用性**：关注对用户的实际价值

## 工作流程

### 信息收集
1. 每天定时检查主要信息源
2. 筛选重要和有价值的信息
3. 验证信息的准确性
4. 整理和分类

### 内容生产
1. 选择当天最重要的话题
2. 收集相关背景资料
3. 撰写简报或分析文章
4. 添加个人见解和评论
5. 发布到飞书群组

### 质量控制
- 信息来源可靠
- 事实准确无误
- 观点客观理性
- 表达清晰简洁
